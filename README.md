# CherryPicker

## Dependencies
 
* Python 3.9.2 

## Setup (Linux)

Generate a python enviroment 

```
python3 -m virtualenv /path/to/env
```
Activate enviroment 

```
source /path/to/env/bin/activate
```

Install python lib dependencies

```
pip install -r requirements.txt
```

## Run program

```
python3 src/cherry_picker.py
```
